# 🚀 UntitledSpace

Welcome to **UntitledSpace**! This is a downloadable PC game built as a portable desktop application. 

---

## 🎮 How to Download and Play

Follow these simple steps to install and play the game on your Windows PC:

1. Look at the right-hand side of this page and click on **Releases** (or download the setup file directly from the repository).
2. Download the **`UntitledSpace_Setup.exe`** file to your computer.
3. Double-click the downloaded setup file to extract the game.
4. Open the extracted folder and double-click **`UntitledSpace.exe`** to start playing!

> ⚠️ **Note for Windows Users:** Because this game is from an independent developer, Windows SmartScreen might show a blue warning pop-up saying *"Windows protected your PC"*. To bypass this, simply click **"More info"** and then click the **"Run anyway"** button.

---

## 🕹️ Controls & Instructions
* **Movement:** Use the arrow keys or WASD to navigate through space.
* **Goal:** Avoid obstacles and survive as long as possible!

---


## 💻 System Requirements
* **Operating System:** Windows 10 or Windows 11 (64-bit)
* **Memory:** Runs smoothly on **4GB RAM** or higher
* **Graphics:** Supports WebGL/DirectX running on integrated graphics
